### rect

    rect(x, y, width, height)

Draws a rectangle with its lower-left corner positioned at `x`, `y` and sized at `width`, `height`.

`x` | `Number` | Horizontal offset
`y` | `Number` | Vertical offset
`width` | `Number` | Width
`height` | `Number` | Height

```Example

====
function Draw()
    rect(0, 0, 50, 50)
end
```
